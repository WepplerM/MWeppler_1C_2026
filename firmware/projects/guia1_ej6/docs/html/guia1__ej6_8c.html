Muestra un número en un display multiplexado usando BCD.

Esta función recibe un número de 32 bits, lo convierte a un arreglo de dígitos decimales (BCD) utilizando la función convertToBcdArray, y luego muestra cada dígito en un display LCD multiplexado utilizando los GPIOs configurados.

Parámetros
dataNúmero de 32 bits a mostrar
digitsCantidad de dígitos a visualizar
bcdGpioVector de GPIOs para los bits BCD (D1-D4)
selGpioVector de GPIOs para selección de dígitos del display
